# Speech Detection 

A modern, high-performance web application for real-time speech-to-text transcription. Built with a premium glassmorphism interface and smooth user experience.

## Features

- **Real-time Transcription**: Uses the Web Speech API for instant, low-latency speech recognition.
- **Premium UI**: Modern glassmorphism theme with `backdrop-filter` effects and deep slate gradients.
- **Manual Control**: Toggle button to start/stop listening for full privacy and control.
- **Status Monitoring**: Visual status pill shows "Listening", "Paused", or "Error" states in real-time.
- **Smooth Animations**: Natural fade-in transitions for new text segments.
- **Auto-scroll**: Transcript area automatically follows your speech for hands-free documentation.
- **Content Filter**: Fun built-in filter that replaces specific words with emojis 

## Getting Started

1. **Clone the repository**:
   ```bash
   https://github.com/hussainsulehri/Speech_Detection.git
   ```
2. **Open the project**:
   Simply open `index.html` in any modern web browser (Chrome or Edge recommended for best Web Speech API support).
3. **Start Speaking**:
   Grant microphone permissions when prompted and watch your voice transform into text!

## Technology Stack

- **HTML5**: Semantic structure and SVG-ready layout.
- **CSS3**: Custom properties (variables), Flexbox, Animations, and Glassmorphism.
- **JavaScript**: Vanilla ES6+ with Web Speech API integration.
- **Typography**: 'Outfit' via Google Fonts.
